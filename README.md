# express-and-gulp

Integrate gulp in nodejs
### Prerequisites
Node version: v8.11.1
NPM version: 5.6.0
### Installing
npm install express-mvc-generator -g
npm install gulp
npm install
## Running the tests
Using gulp in terminal
```
gulp
```
##Check coding covension
```
gulp lint
```
